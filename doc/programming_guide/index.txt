Cocos Programming Guide
========================
:breadcrumb: Programming Guide

The Cocos Programming Guide provides in-depth documentation for writing
applications that use Cocos.  Many topics described here reference the Cocos 
API reference, provided separately.

If this is your first time reading about Cocos, we suggest you start at
`Writing a Cocos application`.
